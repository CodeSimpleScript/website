All code uses short code before all code types, and below is a table of the different types and what they stand for.

|  Code Type	|  Short Code	|
|---	|---	|
|  function	|  f.	|
|  system function	|  s.	|
|  variable	|  v.	|
|  global variable	|  gv.	|
|  posted variable	|  pv.	|
|  session variable	|  sv.	|
|  cookie variable	|  cv.	|
|  url variable	|  uv.	|
|  template	|  t.	|
